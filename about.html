PixelBatch

The quiet manifesto

PixelBatch exists because modern software forgot something basic: your computer is already powerful.

Image tools used to be small, fast, and local. Now they’re web apps stuffed with subscriptions, limits, watermarks, and trackers — charging you monthly for doing work on your own hardware.

This tool runs entirely in your browser. No uploads. No logins. No surveillance. No artificial caps designed to push you into a plan.

Open source isn’t a feature. It’s a minimum standard.

If software can run locally, it should. If it doesn’t need a server, it shouldn’t pretend.

Fuck dark patterns. Fuck rent-seeking SaaS for basic utilities. Build tools people actually own.
